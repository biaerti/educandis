## Jak generować grafiki (Figma AI)

1. Otwórz Figma → utwórz nowy plik
2. Utwórz frame o odpowiednim rozmiarze (1200×630 dla FB, 1080×1080 dla IG)
3. Użyj pluginu **Figma AI** lub **Magician** / **AI Image Generation**
4. Wklej prompt z sekcji "Prompt do Figma AI" z danego pliku posta
5. Dostosuj kolory do brandbooku: `#0f172a`, `#e34234`, `#ffffff`
6. Eksportuj jako PNG 2x

**Alternatywnie:** możesz wkleić prompt bezpośrednio do Midjourney lub DALL-E i dodać do grafiki tekst ręcznie w Figma/Canva.

## Ton głosu (brand voice)

- Profesjonalny, ale bez korporacyjnego żargonu
- Konkretny – liczby, fakty, procenty
- Pomocny – tłumaczymy, a nie sprzedajemy
- Bez zbędnych emotikonek – maksymalnie 2-3 per post
- Nie krzyczysz CAPSLOCKIEM – używasz **pogrubień**
